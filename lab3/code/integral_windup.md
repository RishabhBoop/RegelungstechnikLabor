# Integral Windup
Jedes System hat eine maximale Stellgröße (in unserem Fall max. $\pm$ 10V). Wenn wir eine sehr große Führungsgröße haben, dann kann die Stellgröße nicht weiter steigen als das Maximum (z.B. wird 20V gebraucht aber nur 10V steht zur Verfügung). Dies bedeutet, dass es immer einen Fehler geben wird, bis die Führungsgröße erreicht wird. Aber bei jedem Durchlauf des Reglers wird der Fehler aufsummiert (Integralanteil) und damit die Stellgröße weiter erhöht. Das führt dazu, dass die Stellgröße über das Maximum hinaus ansteigt, was zu einem Überschwingen und Instabilität führen kann. Es braucht eine gewisse Zeit, bis der Fehler wieder auf Null sinkt und die Stellgröße sich anpasst.
